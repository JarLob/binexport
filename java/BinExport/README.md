Build dependencies
------------------

To be installed:

*   Java 11
*   Ghidra 9.1 (https://ghidra-sre.org/releaseNotes_9.1_final.html)
*   Gradle 5.6 or later
    (https://services.gradle.org/distributions/gradle-5.6.3-all.zip)

Gradle will download:

*   Gradle plugin for protobuf
*   Protobuf 3.10.0 for Java
